# 🌐 Projeto Web Responsivo – CP2eJr

## 📌 Sobre o Projeto

Este projeto consiste no desenvolvimento de uma página web responsiva para uma empresa, com foco em usabilidade, organização visual e adaptação a diferentes dispositivos.

O objetivo principal foi criar uma interface moderna e funcional, garantindo uma boa experiência tanto em desktop quanto em dispositivos móveis.

---

## 🚀 Funcionalidades

* Layout totalmente responsivo
* Estrutura organizada com HTML e CSS
* Uso de **Flexbox** e **Grid Layout** para alinhamento e distribuição dos elementos
* Interface intuitiva e de fácil navegação
* Componentes reutilizáveis

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Media Queries

---

## 📱 Responsividade

O projeto foi desenvolvido utilizando **media queries** para adaptação em diferentes tamanhos de tela, garantindo:

* Boa visualização em smartphones
* Ajuste de elementos em tablets
* Layout otimizado para desktop

---

## 🎯 Desafios Enfrentados

Durante o desenvolvimento, alguns desafios importantes foram:

* Organização do layout com múltiplos elementos
* Ajuste fino de espaçamentos
* Manter consistência visual entre diferentes resoluções
* Estruturação do código para facilitar manutenção

---

## 💡 Aprendizados

Mesmo já tendo conhecimento prévio das tecnologias utilizadas, este projeto permitiu aprofundar pontos importantes na prática, como:

* Tomada de decisões entre **Flexbox e Grid** dependendo do contexto do layout
* Organização de estrutura responsiva de forma mais escalável
* Ajustes finos de espaçamento e alinhamento em diferentes resoluções
* Pensamento mais crítico sobre experiência do usuário em diferentes dispositivos
* Estruturação de código visando manutenção e clareza

Mais do que aprender conceitos novos, o projeto contribuiu para consolidar boas práticas no desenvolvimento front-end.

---

## 📂 Como Executar o Projeto

1. Clone o repositório:

```bash
git clone https://github.com/LuisHenriqueCGJunior/Cp2eJr.git
```

2. Acesse a pasta do projeto:

```bash
cd Cp2eJr
```

3. Abra o arquivo `index.html` no navegador
